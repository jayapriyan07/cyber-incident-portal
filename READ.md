# Cyber Incident Reporting Portal

Problem Statement ID: SIH25183  
Organization: Ministry of Defence (MoD)

## Project Description
This project is a web portal that allows users to report cyber incidents 
such as phishing, malware attacks, and suspicious activities.

## Features
- User Registration
- Login System
- Cyber Incident Reporting
- Admin Page

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub

## How to Run
1. Download the project
2. Open index.html in browser
